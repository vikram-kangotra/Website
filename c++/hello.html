Say Hello to the world!

Have you been thinking about making your application, But don't know where to start? Worry not. Today we will start the journey by making the world-famous hello world program in C++.

Prerequisite

  1. Must know how to operate a computer.
  2. Being comfortable in working with a terminal.
  3. That's it :)

What we need?

In order to make a C++ application, the most basic thing that we need is a compiler.

Compiler

A compiler is a program that converts human-readable code (C/C++) into machine-readable code (machine code). It acts as a translator. Jump here to read more.

Although there are many compilers out in the market, I use g++. Also, It's free.

Now a lot of people might not agree with me, but I have got a strong faith that one should refrain from using an IDE at the beginning stage and try to get comfortable with using a terminal. The Sooner, the better.

Having said that, I expect you have already setup g++. If not, search for some article on the web. There are a lot.

If you feel you're not ready enough for using a terminal, Don't worry. Just download codeblocks. It's an IDE. And see how to use it.

Hello?

without further ado, let's start.

Fire up your favorite text editor and save it as what_ever_you_want.cpp. I'd save it as main.cpp

Now write.

  1. int main() {
  2. return 0;
  3. }

This is the minimalistic code that one could write.

Every C++ program or any program in general needs to start somewhere. In C++, it is main. Everytime you run the program, the machine code will be loaded and the OS will call the main function and then the execution will start. More on that later.

Now add the below line under the first line.

  1. std::cout << "Hello World!";

std::cout is the C++ way of printing something on screen.

But C++ doesn't what std::cout is as such. And we need to tell it where to look for its meaning.

Add the next line at start of the program.

  1. #include <iostream>

iostream is the file which contains the meaning for std::cout.

Let's see what our program looks like

  1. #include <iostream>
  2. int main() {
  3. std::cout << "Hello World!";
  4. return 0;
  5. }
